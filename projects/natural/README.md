# Natural – Natural Language Processing Toolkit for Node.js (Contribution Fork)

### 📌 About
Natural is a general natural language facility for Node.js. It provides a wide range of NLP tools such as tokenizers, stemmers, classifiers, phonetics, tf-idf, WordNet, and more.

This fork includes additional contributions and enhancements to improve accuracy, maintainability, and modern Node.js support.

---

### ✨ Features
- Tokenization (word, sentence, regex)
- Stemming (Porter, Lancaster)
- Classification (Naive Bayes, Logistic Regression)
- Phonetics (Soundex, Metaphone)
- Tf-idf term scoring
- WordNet integration
- Spellcheck, language detection
- N-Grams and other NLP utilities

---

### 🛠 Stack
- **Core Language:** JavaScript (Node.js)
- **Testing:** Mocha, Chai
- **Build & Tools:** NPM, Browserify
